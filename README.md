# JSON Spark 🌟
A clean, modern, and highly responsive client-side JSON Formatter and Validator utility platform.

## Live Demo
👉 [Click here to use the tool live on Vercel](https://json-formatter-weld.vercel.app/)

## Features
- Client-side text parsing and interactive minification selectors.
- Recursive DOM node tree structural rendering with collapsible nodes.
- Real-time syntax validation with dynamic error index feedback.
